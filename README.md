# Taha Shams Portfolio

A modern personal portfolio website built with HTML, CSS, and JavaScript.

## Overview

This site presents:
- A hero section with profile image and intro
- An About section with skills
- A Projects section with direct GitHub links
- A Contact section with email call-to-action

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts (DM Serif Display, DM Mono, Outfit)

## Project Structure

- `portfolio.html` - Main page structure and content
- `styles.css` - Full site styling, layout, and responsive design
- `script.js` - Scroll navigation effect and section reveal animations
- `profile.jpg` - Profile image used in hero section

## Features

- Responsive single-page layout
- Smooth scrolling navigation
- Sticky nav style change on scroll
- Reveal-on-scroll section animation
- Clickable project cards opening GitHub repositories in new tabs
- Keyboard-accessible focus style for project links
